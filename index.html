Core Modules
🔍
Modular Scanning Pipeline
Orchestrates parallel scan phases across customizable modules. Supports plugin-based architecture — add recon, enumeration, or exploit modules without touching core logic. CIDR range input, host discovery, OS fingerprinting via Nmap integration.
🌐
HTTP Security Analysis
Deep inspection of web services — analyzes response headers (HSTS, CSP, X-Frame-Options, CORS policy), detects misconfigurations, tests for open redirects, directory traversal, and checks TLS/SSL certificate validity and cipher suite strength.
🧬
CVE Detection Engine
Maps discovered service banners and version strings against a local NVD-synced CVE database. Scores each finding using CVSS v3.1 severity, flags exploitable entries, and cross-references with ExploitDB for available PoC code.
📋
Report Generation
Produces structured PDF and JSON reports with executive summary, per-host findings, severity matrix, and remediation recommendations. Reports include timestamps, scan metadata, and evidence screenshots where applicable.
